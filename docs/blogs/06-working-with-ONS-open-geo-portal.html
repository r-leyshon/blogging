Introduction

This tutorial is for programmers familiar with Python but perhaps less familiar with the Python requests package or ArcGIS [1].

The Scenario

You would like to use python to programattically ingest data from the Office for National Statistics (ONS) Open Geography Portal. This tutorial aims to help you do this, working with the 2021 LSOA boundaries, the essential features and quirks of the ArcGIS REST API will be explored.

What you’ll need:

  • A permissive firewall (ask to whitelist the domain “https://geoportal.statistics.gov.uk/”)
  • Python package manager (eg pip)
  • Python environment manager (eg venv, poetry etc)
  • Python requirements:
requirements.txt
folium
geopandas
mapclassify
matplotlib
pandas
requests

Tutorial

Setting Things Up

  1. Create a new directory with a requirements file as shown above.
  2. Create a new virtual environment.
  3. pip install -r requirements.txt
  4. Create a file called get_data.py or whatever you would like to call it. The rest of the tutorial will work with this file.
  5. Add the following lines to the top of get_data.py and run them, this ensures that you have the dependencies needed to run the rest of the code:
get_data.py
import requests
import geopandas as gpd
import pandas as pd
